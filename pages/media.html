Media & Embedding

Embed images, audio, and video in your HTML documents

<img> - Images

The most basic way to embed images in HTML.

<img src="image.jpg" alt="Description of image" width="400" height="300">

Image Attributes

AttributePurpose
srcImage file URL (required)
altAlternative text for accessibility
widthImage width in pixels
heightImage height in pixels
titleTooltip on hover

<picture> - Responsive Images

<picture>
  <source media="(max-width: 600px)" srcset="small.jpg">
  <source media="(max-width: 1000px)" srcset="medium.jpg">
  <img src="large.jpg" alt="Responsive image">
</picture>

<audio> - Audio Files

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  <source src="audio.ogg" type="audio/ogg">
  Your browser doesn't support audio element.
</audio>

<video> - Video Files

<video width="320" height="240" controls>
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  Your browser doesn't support video element.
</video>

Video/Audio Attributes

AttributePurpose
controlsShow playback controls
autoplayStart playing automatically
loopRepeat when finished
mutedStart without sound
preloadLoad strategy (auto, metadata, none)

<figure> & <figcaption>

Group images or media with captions:

<figure>
  <img src="image.jpg" alt="Scenic landscape">
  <figcaption>A beautiful mountain landscape</figcaption>
</figure>

Best Practices

  • Always include alt text for images
  • Optimize images for web (compress, correct format)
  • Use responsive images for different screen sizes
  • Provide multiple source formats for video/audio
  • Use <figure> for images with captions
  • Consider performance implications of large media files

Accessibility Guidelines

  • Provide descriptive alt text
  • Include captions for videos
  • Provide transcripts for audio
  • Don't rely on color alone to convey information
  • Make media controls keyboard accessible
