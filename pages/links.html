Links & Navigation

Master the anchor element and create effective navigation structures

The <a> Element

The anchor element creates hyperlinks to other pages, files, or locations within the same page.

<a href="https://example.com">Click here</a>

Essential Attributes

AttributeDescription
hrefThe URL the link points to (required)
targetWhere to open the link (_blank, _self, _parent, _top)
titleTooltip text on hover
relRelationship between documents (nofollow, external, etc.)
downloadPrompts download instead of navigation

Types of Links

External Links

<a href="https://www.example.com">Visit Example</a>

Internal Links

<a href="/about/team.html">Our Team</a>

Anchor Links (Bookmarks)

<a href="#section-id">Jump to Section</a>

<section id="section-id">
  <h2>Section Title</h2>
</section>

Email Links

<a href="mailto:info@example.com">Send Email</a>

Phone Links

<a href="tel:+1234567890">Call Us</a>

Best Practices

  • Use descriptive link text that explains where the link goes
  • Avoid "Click here" or "Read more" as link text
  • Use target="_blank" sparingly for external links
  • Include rel="noopener noreferrer" when using target="_blank"
  • Ensure links are keyboard accessible
  • Provide visual focus indicators for accessibility

Navigation Example

<nav>
  <ul>
    <li><a href="/">Home</a></li>
    <li><a href="/about">About</a></li>
    <li><a href="/contact">Contact</a></li>
  </ul>
</nav>

Accessibility Guidelines

  • Use meaningful link text
  • Indicate external links with an icon or text
  • Color should not be the only indicator of a link
  • Ensure sufficient color contrast
  • Include title attributes for additional context
