Document Structure

Master the foundation of every HTML document with essential structural elements

Introduction

The HTML document structure elements form the backbone of every webpage. These elements define the basic layout and organization of your HTML document, including the root element, metadata, and content areas.

<html> Element

The root element that wraps all other HTML elements.

Syntax

<!DOCTYPE html>
<html lang="en">
  <!-- Content goes here -->
</html>

Attributes

AttributeDescription
langSpecifies the language of the document
dirSets the text direction (ltr or rtl)

Browser Output

Note: The <html> tag itself is not visible on the page. It's a container for all other HTML elements.

<head> Element

Contains metadata and links to scripts and stylesheets. Information in the <head> is not displayed on the page.

Syntax

<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
  <link rel="stylesheet" href="style.css">
</head>

Common Head Elements

ElementPurpose
<meta>Defines metadata about the document
<title>Defines the title shown in browser tab
<link>Links to external resources like CSS
<style>Defines internal CSS styles
<script>Includes JavaScript code

<body> Element

Contains all the visible page content that appears in the browser window.

Syntax

<body>
  <h1>Welcome to My Page</h1>
  <p>This content is visible in the browser.</p>
</body>

Attributes

AttributeDescription
onloadScript to run when page loads
onunloadScript to run when page unloads

Best Practices

  • Always include DOCTYPE declaration at the beginning
  • Set the lang attribute on the <html> element for accessibility
  • Include meta charset in the head for proper text encoding
  • Set a descriptive title for each page
  • Include viewport meta tag for responsive design
  • Place scripts at the end of the body for better performance

Common Mistakes

❌ Missing DOCTYPE

<html>
  <head></head>
  <body></body>
</html>

Problem: Without DOCTYPE, the browser may render in quirks mode.

✅ Correct DOCTYPE

<!DOCTYPE html>
<html lang="en">
  <head></head>
  <body></body>
</html>

Accessibility

  • Use lang attribute to specify document language
  • Provide meaningful page titles
  • Use proper heading hierarchy (h1-h6)
  • Include alt text for images
  • Use semantic HTML elements
